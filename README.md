Andes Retail Group – Dashboard de Desempeño Comercial (Power BI)
- Descripción del Proyecto
Este proyecto tiene como objetivo diseñar y desarrollar un dashboard interactivo en Power BI que permita analizar el desempeño comercial de Andes Retail Group durante los años 2024 y 2025.
El enfoque principal es transformar datos transaccionales en insights accionables, facilitando la toma de decisiones estratégicas mediante visualizaciones claras y una narrativa ejecutiva.

- Objetivos
Analizar la evolución de ingresos en el tiempo
Identificar diferencias por:
País
Categoría de producto
Segmento de cliente
Estacionalidad
Evaluar la rentabilidad del negocio (Ingresos vs Costos)
Detectar oportunidades de mejora comercial

- Fuente de Datos
Archivo utilizado:
  Andes_Retail_Group_2024_2025.xlsx

- Contiene información de:
  Ventas
  Clientes
  Productos
  Geografía
  Fechas
